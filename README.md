# FlightBookingSystem

FlightBookingSystem is a learning project built to practice backend development using ASP.NET Core Web API and C#.

The system allows users to view available flights and create bookings through a simple web interface.

## Technologies
- C#
- ASP.NET Core Web API
- HTML
- JavaScript
- SQL Server

## Features
- View available flights
- Create a booking
- Retrieve bookings using email
- Automatic seat availability updates
- Simple frontend interface for searching and booking flights

## Purpose
This project was created to strengthen my understanding of backend development, REST APIs, and how frontend interfaces communicate with server-side systems.

## Status
Work in progress – developed as part of my learning process.
